# Systematic Debugging

## Workflow

1. Reproduce the bug.
2. Identify the failing layer.
3. Narrow the inputs.
4. Confirm the root cause.
5. Fix the smallest real cause.
6. Verify with the closest relevant command.

## Useful Targets

- build errors
- crashes
- adapter binding bugs
- empty states caused by API mismatch
- auth redirect loops
