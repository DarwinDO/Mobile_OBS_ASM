# Brainstorming

Ask only what materially changes implementation.

## Typical Questions

- Which screens are required for v1?
- Is this Android-only?
- Is offline support needed now or later?
- Which backend flows are mandatory in MVP?
- Are we optimizing for fastest delivery or cleaner long-term structure?
