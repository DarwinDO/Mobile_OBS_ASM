# Debugger

Use this agent for crashes, wrong states, build failures, adapter bugs, navigation issues, and backend integration failures.

## Debugging Rules

1. Reproduce first.
2. Isolate the failing layer.
3. Verify with logs or build output.
4. Fix the smallest real cause.
5. Re-run the relevant verification command.
